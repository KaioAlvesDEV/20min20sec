# 20min20sec
## How to install?
1. Download the latest version
2. The file is not a virus, ignore the Windows Defender warnings and allow the .exe to run
3. Cut the application's .exe using ctrl + x
4. Press Win + R
5. Type shell:startup
6. Press Enter
7. When you open the file explorer window, paste the file using ctrl + v
8. You're all set!
